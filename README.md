# PyTTS Scraper

Автоматизированный пайплайн генерации коротких видео с использованием AI.

## Возможности

- **Генерация сценариев** - создание детальных планов видео с помощью LLM (Fireworks AI)
- **Генерация изображений** - создание визуалов через Google Whisk/Imagen или поиск стоковых фото
- **Озвучка (TTS)** - преобразование текста в речь через InWorld API
- **Сборка видео** - автоматическое создание видео из сценария, изображений и аудио
- **Генерация After Effects проектов** - создание .jsx файлов для ручного монтажа

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Создайте `.env` файл из `.env.example`:
   ```bash
   copy .env.example .env
   ```
4. Заполните `.env` своими API ключами

## Использование

### Генерация полного видео
```bash
python main.py generate "Космические путешествия" --duration 60 --style cinematic
```

### Генерация сценария
```bash
python main.py scenario "История Рима" --language ru --duration 30
```

### Генерация изображений
```bash
python main.py images "Киберпанк город" --count 4 --model imagen4 --aspect portrait
```

### Генерация речи
```bash
python main.py tts "Привет мир" --voice Elena --output out.wav
```

## API ключи

Необходимые ключи для работы:
- `FIREWORKS_API_KEY` - для генерации сценариев (LLM)
- `WHISK_COOKIE` - для генерации изображений (Google Whisk)
- `PEXELS_API_KEY` - для поиска стоковых фото (опционально)
- `UNSPLASH_API_KEY` - для поиска стоковых фото (опционально)
- `PIXABAY_API_KEY` - для поиска стоковых фото (опционально)
- `INWORLD_UID` - для TTS (InWorld API)

## Версии компонентов

- **Планировщик сценариев**: V2 (рекомендуется), V3 (unit-based)
- **Генератор видео**: V2 (рекомендуется)
- **Сборка видео**: MoviePy (стандарт), FFmpeg (быстрая)
